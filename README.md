# How to Start

Install 
1. Node JS
2. Java SDK & JAVA_HOME
3. Android Studio & ANDROID_HOME
4. Appium Desktop
5. Install package.json with command

```
  npm install
```

How to run
1. Connect to your device android or IOS
2. Setup desired capabilities in file wdio.conf.js
3. Run test
```
  npm run wdio
```
